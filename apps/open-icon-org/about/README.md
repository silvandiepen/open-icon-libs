# About Open Icon

Open Icon is built around practical SVGs: clean source files, stable names, and runtime styling that stays outside the artwork.

Use this section when you want the rules behind the icon system instead of just the catalog.

- Read the [Philosophy](./philosophy/) page for the design principles behind the set.
- Read the [Icon Authoring Guide](./authoring/) for source SVG conventions and transform behavior.
- Read the [Release Notes](./release-notes/) page for published changes.
